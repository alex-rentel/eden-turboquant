# mlx-turboquant

**Near-optimal KV cache quantization for MLX on Apple Silicon.**

Faithful implementation of [TurboQuant: Online Vector Quantization with Near-optimal Distortion Rate](https://arxiv.org/abs/2504.19874) (Zandieh, Daliri, Hadian, Mirrokni — Google Research / DeepMind / NYU, ICLR 2026).
